# Frontend del Juego de Bingo

Este es el frontend de la aplicación de Bingo en tiempo real, desarrollado con React.

## Requisitos Previos

- Node.js (versión 14 o superior)
- npm (incluido con Node.js)

## Instalación

1. Instalar las dependencias:
```bash
npm install
```

2. Iniciar la aplicación en modo desarrollo:
```bash
npm start
```

La aplicación se abrirá automáticamente en tu navegador en `http://localhost:3000`.

## Características

- Cartón de bingo interactivo
- Visualización de balotas en tiempo real
- Verificación automática de victoria
- Interfaz moderna y responsiva

## Estructura del Proyecto

```
src/
├── components/      # Componentes reutilizables
├── App.jsx         # Componente principal
└── index.js        # Punto de entrada
```

## Tecnologías Utilizadas

- React 18
- Styled Components
- Socket.IO Client
